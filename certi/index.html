安装组件

1. 安装 Loaders

加密版本请先安装 loaders

平台 下载地址
linux ixed.7.1.lin
linux ixed.7.2.lin
windows ixed.7.1.win
windows ixed.7.2.win
MacOs ixed.7.1.dar
MacOs ixed.7.2.dar

以 Linux 配置为例

部署ixed.7.1.lin

php/lib/php/extensions/no-debug-non-zts-20131226/ixed.7.1.lin

配置php.ini

extension = ixed.7.1.lin

重启 nginx/php-fpm 完成配置
